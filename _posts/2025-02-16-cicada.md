Foothold involves using `netexec`, `ldapdomaindump` and Impacket scripts to retrieve multiple sets of credentials from SMB and LDAP, and eventually getting a shell with WinRM. ### Creating Shadow Copy of C:\

I’ll first create the following `.dsh` file, as shown in [this walkthrough](https://www.hackingarticles.in/windows-privilege-escalation-sebackupprivilege/):
```
set context persistent nowriters
add volume c: alias cdrive
create
expose %cdrive% e:
```

Running it with `diskshadow` should create a copy of the C drive in `E:\`.
